# Project Overview
This project addresses the high complexity and volatility of modern financial markets by replacing traditional analysis with a data-driven framework. Conducted as a collaborative effort, the study utilizes a dataset of five major U.S. companies—AAPL, JPM, JNJ, XOM, and PG—with historical data spanning from 2017 to 2025. The goal was to move beyond simple historical analysis to provide diverse methods for predictive signals and superior portfolio balancing.

Technical Modules
1. Predictive Modeling (Bayesian Networks)Objective: Forecast short-term stock returns by understanding market dynamics through technical indicators.Implementation: Developed a probabilistic graphical model using the HillSearchClimb algorithm with BIC for each company.Results: Achieved predictive accuracies between 73% and 76%, validating the model's ability to capture intricate dependencies in financial data.
2. Feature Selection (Genetic Algorithm & Regression)Objective: Identify and validate the most influential financial indicators driving performance.Implementation: Employed a hybrid approach combining Genetic Algorithms (GA) with Ordinary Least Squares (OLS) Regression to find optimal predictive factor combinations.Results: The selected factor sets demonstrated strong explanatory power, reaching an $R^2$ as high as 0.761 (JPM).
3. Portfolio Optimization & BacktestingObjective: Construct investment portfolios that maximize returns while minimizing risk using computational methods.Strategies: Evaluated various models including Markowitz Mean-Variance, Risk Parity, CVaR, and Robust Optimization (Ledoit-Wolf Shrinkage).Results: In out-of-sample testing on unseen data, the Genetic Algorithm and Robust Shrinkage methods achieved the highest Sharpe and Sortino Ratios.
